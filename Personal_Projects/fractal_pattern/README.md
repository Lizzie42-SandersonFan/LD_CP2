# Sierpinski Triangle Generator
***
![Image of code while runnign](location of image in folders)
This program will let you generate a Sierpinski Triangle with a depth of 1 to 5.

## How to use
***
1. Go to the file named "main.py"
2. Go to the top and hit the Run Button
3. It will ask for depth and colors before drawing via Turtle
