# Sierpinski Triangle Generator
***
This program will let you generate a Sierpinski Triangle with a depth of 1 to 5.

## How to use
***
1. Go to the file named "main.py"
2. Go to the top and hit the Run Button
3. It will ask for depth and colors before drawing the triangle via Turtle

## Project Features
***
- Main feature is being able to draw the Serpinski Triangle with Turtle 🐢
- Line color and Background color cannot be the same ⚪⚫
- Full choice of colors by using heaxcodes! 🔴🟠🟡🟢🔵🟣
